Jquery Multi Select Plugin v1.0

Features

  1. Pick multi level category swiftly
  2. Unlimited category level
  3. Super easy to implement
  4. Customizable patterned value

File structure

  1. index.html
  2. js

Step 3

Options

Preferably your own defined delimiter can be used, the default delimiter is "-" you can change to your own by something like this $('#carform').multiSelect(':') in this case your delimiter will be ":"

How the plugin works

The plugin will look for select element in your form. Treat each one as a category level. So if you your form has five select elements, the plugin will think you want to have 5 levels category.
